Current

Browser Extensions

  • Bitwarden: My password manager.
  • Minim: A distraction-free new tab page.
  • SocialFocus: Removes distractions from sites.

Hardware

  • Kobo Clara 2E E-Reader: Life changing! Reading in it is a pleasure.
  • Lenovo Yoga 9 Laptop: I adore how quiet and ergonomic the keyboard is.
  • Logitech MX Anywhere 25 Mouse: Fast and comfortable, as needed.
  • Samsung 21+ Phone: Love how big it is. It's just a glorified tablet.
  • Soundpeats Earphones: Excellent audio quality!

Programming

  • Apache: Web server capabilities.
  • Git: Tracking changes made to my sites and notes.
  • PHP: Lightweight scripts for my sites.

Services

  • Cloudflare: my DNS management.
  • Fastmail: My email provider.
  • Libby: More books from the library!
  • Pinterest: The strangest, but most inspiring inputs.
  • Webmention.io: Allows my three sites to receive webmentions.
  • Wikipedia: It satisfies my endless curiosity.

Software

  • Edge: Does what I need it to, blazingly fast.
  • ExpressVPN: An internet necessity.
  • Flow Launcher: My entire computer is operable from my keyboard. Extraordinary.
  • Krita: Digital drawing.
  • Mega: File storage and sync.
  • Obsidian: All my notes. Makes life much easier. I have the Catppuccin Mocha theme, plus the Git and Style Settings plugins.
  • Slack: Familial and work chat.
  • Terminal: For all my work in the command line.
  • VSCodium: My code editor with the Catppuccin Mocha theme.
