Uses

Published: 29 Jun 2024 | Updated: 27 Feb 2025

My contribution to /uses, compiled by Wes Bos. See more folks' here. These are the tech tools I use across Android and Windows.

The programing languages I use are basic CSS, HTML, Markdown, and XML.

Browser Extensions

  • Bitwarden: My password manager.
  • Minim: A distraction-free new tab page.
  • Unhook: Calmer layouts for YouTube.

Hardware

  • Kobo Clara 2E E-Reader: Life changing! Reading in it is a pleasure.
  • Lenovo Yoga 9 Laptop: I adore how quiet and ergonomic the keyboard is.
  • Logitech MX Anywhere 25 Mouse: Fast and comfortable, as needed.
  • Samsung 21+ Phone: Love how big it is. It's just a glorified tablet.
  • Soundpeats Earphones: Excellent audio quality!

Services

  • Borrowbox: Books from the library!
  • Fastmail: My email provider.
  • IngramSpark: For book publishing.
  • Libby: More books from the library!
  • Pinterest: The strangest, but most inspiring inputs.
  • Wikipedia: It satisfies my endless curiosity.

Software

  • Edge: Does what I need it to, blazingly fast.
  • ExpressVPN: An internet necessity.
  • Filezilla: Easy uploads for my site files to my hosting.
  • Flow Launcher: My entire computer is operable from my keyboard. Extraordinary.
  • Krita: Digital drawing.
  • Media Player: For listening to music on repeat
  • Mega: File storage and sync.
  • NotePad: I write all my code in here.
  • Obsidian: All my notes. Makes life much easier.
  • Slack: Familial and work chat.

Previously

Browser Extensions

  • Cold Turkey Blocker: The accompanying extension for the software.
  • Get RSS Feed URL: As the name says! Now I just use a bookmarklet.. 2024-2025.
  • Wayback Machine: Keeping my site archived. Now I just use a bookmarklet. 2024-2025.

Hardware

  • Audeo Trek Earphones: Surprisingly good for a supermarket purchase. 2022-2024.
  • Generic Monitor: Tried using a second screen, but couldn't get into it. 2023-2024.
  • Logitech Keyboard: Don't use it because I don't have an external monitor. 2023-2024

Services

  • Github: Manual open-sourcing my site was too much work. 2024.

Software

  • Beeftext: A snippet expander. There were too many programs on my computer. 2023-2024
  • Cold Turkey Blocker: Takes back my focus. I use the hosts file on Windows instead. 2024-2025
  • Dropbox: Where I stored my files. Replaced it with Mega. 2024.
